<h1>#Flask Sign-Up & Login System</h1>

<h2>A simple and clean authentication system built using Python Flask, HTML/CSS/JS, and SQLite.
It allows users to create an account with a username, email, and password, stores everything in a secure database, and lets them log in later by matching their credentials.</h2>

# Features

User Sign-Up with username, email, and password

Login system with validation

SQLite database to store user details

Duplicate email/username check

Flash messages for alerts

Clean front-end UI using HTML, CSS & JavaScript

Simple and beginner-friendly project structure

# Tech Stack

1/Python (Flask)

2/ HTML5

3/ CSS3

4/ JavaScript

5/ SQLite3

# How It Works

1/ User opens the Sign-Up page

2/ Enters details → Data stored in SQLite

3/ On login, Flask compares:

  - username/email

  - password

If valid → User gets logged in successfully

Else → User sees appropriate error message

# How to Run the Project

1/ Clone the repo:

git clone https://github.com/Samin-Saikia/Flask-Sign-Up-Login-System.git


2/ Install dependencies:

pip install flask
pip install sqlalchemy


3/Run the app:

python app.py


4/Open in browser:

http://127.0.0.1:5000/
